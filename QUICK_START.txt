WSL2 IS ALREADY INSTALLED ✓

Follow these 3 simple steps to launch Agent X web interface:

========================================================================
STEP 1: OPEN UBUNTU (NOT POWERSHELL)
========================================================================

Press Windows key, type "Ubuntu", click the Ubuntu app.

You'll see a green terminal that looks like:
    bob@LAPTOP:~$

========================================================================
STEP 2: DEPLOY AGENT X
========================================================================

In the Ubuntu terminal, copy and paste this entire block:

curl -fsSL https://raw.githubusercontent.com/opusmax422-dot/-LONGO-AGENT-X---DEAD-TO-RIGHTS-LOOK-OUT-LONGO-FIVE-EYES-WATCHING-GOVERNMENT-CONSPIRACY-EXPOSED-/main/DEPLOY_TO_LOCAL_PC.sh | bash

OR manually:

cd ~
git clone https://github.com/opusmax422-dot/-LONGO-AGENT-X---DEAD-TO-RIGHTS-LOOK-OUT-LONGO-FIVE-EYES-WATCHING-GOVERNMENT-CONSPIRACY-EXPOSED-.git
cd *LONGO*
bash DEPLOY_TO_LOCAL_PC.sh

This checks dependencies and sets everything up.

========================================================================
STEP 3: START AGENT X
========================================================================

bash start-agent-x.sh

Browser will open automatically to http://localhost:8080

========================================================================
